# Estructura del Proyecto

Este documento describe la estructura de directorios del proyecto después de la consolidación realizada en la rama `dev`.

## Árbol de directorios

```text
src/
├── analisis/
├── core/
├── diferencias/
├── edo/
├── errors/
├── integracion/
├── interpolacion/
├── lineales/
├── no-lineales/
├── polinomios/
├── root/
└── utils/
```

## Descripción de directorios

### analisis/
Funciones para análisis numérico y procesamiento de resultados.

### core/
Componentes base compartidos por los distintos métodos numéricos.

### diferencias/
Implementa métodos de diferencias finitas.

### edo/
Métodos para resolver ecuaciones diferenciales ordinarias.

### errors/
Definición y manejo de errores personalizados.

### integracion/
Métodos de integración numérica.

### interpolacion/
Métodos de interpolación de datos.

### lineales/
Métodos para resolver sistemas de ecuaciones lineales.

### no-lineales/
Métodos para resolver ecuaciones no lineales.

### polinomios/
Herramientas para evaluación y manipulación de polinomios.

### root/
Métodos para búsqueda de raíces de funciones.

### utils/
Funciones auxiliares reutilizables utilizadas por todo el proyecto.
```
